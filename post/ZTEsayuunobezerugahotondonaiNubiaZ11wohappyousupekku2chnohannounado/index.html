ZTE､左右のベゼルがほとんどない｢Nubia Z11｣を発表 ｰ スペック・2chの反応など

2016.6.28 remove_red_eye4082

ZTEは6月28日、「Nubia」ブランドのAndroidスマートフォン「Nubia Z11」を発表しました。

「Nubia Z11」は、5.5インチ（1920 × 1080）、Snapdragon 820、RAM 4GB / 6GB、ストレージ 64GB / 128GB、背面に1600万画素カメラ（IMX298・光学式手ブレ補正・PDAF）、前面に800万画素カメラ、USB Type-C、指紋リーダー、3000mAhバッテリー（QC3.0）、Android 6.0+Nubia UI 4.0などを搭載。


通信機能はFDD-LTE（B1 / B3 / B7）、TD-LTE（B38 / B39 / B40 / B41）、TD-SCDMA（B34 / B39）、WCDMA（850 / 900 / 1900 / 2100）、GSM（850 / 900 / 1800 / 1900）、VoLTE、Wi-Fi a / b / g / n / ac、Bluetooth 4.1、NFCをサポート。SIMスロットはNanoSIM2基（片方はmicroSDカードと兼用）

筐体のサイズは151.8 x 72.3 x 7.5mm、質量は162g。カラバリはゴールド、グレー、シルバー、コーヒーゴールド。

価格は 4GB RAM + 64GB ROMモデルが2499元（約3万8500円）、6GB RAM + 128GB ROMモデルが3499元（5万3900円）。7月6日に中国で発売される予定です。

Source： ZTE
