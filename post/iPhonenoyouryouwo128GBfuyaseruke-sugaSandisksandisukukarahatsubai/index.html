iPhoneの容量を128GB増やせるケースがSandisk(サンディスク)から発売

2016.6.27 ニュース remove_red_eye3187


画像引用元： Meet the new SanDisk® iXpand Memory Case 以下、同じ

iPhoneのストレージ容量を簡単に増量させられるケースが発売しました。

それが、『 iXPAND MEMORY CASE 』というiPhoneケースです。SDカードや外付けSSDなどで有名な Sandisk（サンディスク） が手がけています。

写真や動画をたくさん保存しているから、残りの容量が不安という方は要チェックですよ！

最大128GB追加できるiPhoneケース

iXPAND MEMORY CASEは、iPhone 6/6S用のケースです。容量は 32GB・64GB・128GB の3種類あり、空き容量などにあわせて選べますよ。

ケース上部が外れるようになっており、iPhoneを挿し込むように装着します。バンパーを装着する感覚に近いでしょう。


自動バックアップ機能やパスワード設定もできる

写真をたくさん撮っていると、知らぬ間に容量が圧迫されていた...。ということがありますよね。

しかたなくその場で写真を削除して容量を空けることも...。でもiXPAND MEMORY CASEをつけていれば、削除せず写真を移し替えることができます。

また、アプリと同期して自動でバックアップを取ってくれる機能もあるんです。


それに加え、ケースに保存したデータはパスワードで保護することができるので、セキュリティ面も安心！

別売りのバッテリーパックを装着できる

iXPAND MEMORY CASEは、メモリだけでなく追加でバッテリーを搭載することもできるのです。

バッテリーは別売りですが、メモリ＋バッテリーが足せるのは旅行や出張など長時間iPhoneを持ち歩く環境では確実に役立つでしょう。


iXPAND MEMORY CASEについての詳細は動画でも確認できますよ。

iPhoneを買うときって容量がどれくらい必要かわからないことが多いですよね。もし、16GBや64GBを購入して、容量不足に悩んでいるなら、チェックしてみてください。
